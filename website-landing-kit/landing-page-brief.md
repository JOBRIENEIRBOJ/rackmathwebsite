# Rack Math Landing Page Brief

## Objective

Improve the Rack Math website homepage so a new lifter immediately understands
what the app does and has a clear path into the product.

Positioning:
Rack Math is a barbell plate calculator, workout tracker, and program companion
for lifters who want less friction between the plan and the bar.

## Audience

- Lifters calculating plates during workouts
- People following barbell, strength, or hypertrophy programs
- Coaches or users importing workouts from written plans
- Gym users who want quick mobile-first workout logging

## First-Viewport Message

Headline:
Load the right plates. Track every set.

Supporting copy:
Rack Math turns planned lifts into clear plate math, workout sessions, PRs, and
progress insights, so you can stay focused under the bar.

Primary CTA:
Open Rack Math

Secondary CTA:
See how it works

## Page Structure

1. Hero
   - Rack Math logo
   - headline
   - short value statement
   - primary and secondary CTA
   - app screenshot or phone mockup
   - three compact value chips

2. Product Proof
   - Plate calculator
   - Program sessions
   - Progress tracking

3. Workflow
   - Enter or import workout
   - See exact plates per set
   - Complete sets and build history
   - Review PRs, muscles, and insights

4. Premium Value
   - Keep this concise and tied to outcomes
   - Cloud sync, imported programs, deeper insights, advanced tracking

5. FAQ
   - Is it free?
   - Does it work on mobile?
   - Can I use kg and lb?
   - Can I import workouts?

6. Final CTA
   - Repeat primary CTA
   - Include install/app language if the hosted app supports PWA install

## Design Direction

- App-first, not brochure-first.
- Use Rack Math blue as the main action color.
- Use real app imagery where possible.
- Keep cards tight, with small radius and dense useful content.
- Avoid oversized decorative hero sections that delay the product.
- Prioritize mobile layout first.

## SEO Targets

Primary title:
Rack Math | Barbell Plate Calculator & Workout Tracker

Primary description:
Rack Math helps lifters calculate barbell plates, follow workouts, import
programs, log sets, and track PRs from a fast mobile-first app.

Keyword themes:
- barbell plate calculator
- workout tracker
- weightlifting app
- plate math
- strength training tracker
- program workout tracker
